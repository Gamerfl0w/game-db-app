<h1 align="center">A Game Database App. Made with React Native and NativeWind/Tailwind CSS</h1>
<h3>Features</h3>
<ul>
  <li>Fetch Data from RAWG API</li>
  <li>View Game Details</li>
  <li>Image viewer with gestures enabled</li>
  <li>Search Games</li>
</ul>
